C++ 信号量的实现

发表于 2022-02-25 更新于 2022-04-08 分类于 C++

C++没有提供信号量，可以通过标准库的mutex、unique_lock和condition_variable实现信号量机制

Mutex类

  • Mutex又称互斥量，C++11中与Mutex相关的类（包括锁类型Lock类）和函数都声明在mutex头文件中

  • 定义与初始化

  • lock()成员，调用线程锁住该互斥量

  • unlock()成员，调用线程释放对互斥量的所有权

  • try_lock()成员，尝试锁住互斥量，若被其他线程占有也不会阻塞。返回false代表互斥量被其他线程占有

unique_lock类

  • 构造函数接受一个mutex，将其锁定
  • 析构函数将mutex解锁

条件变量类

  • 通过上述互斥量和条件变量，可以构造管程

  • 定义在头文件condition_variable中

  • 定义与初始化

  • 条件变量的wait成员

  • 条件变量的notify_one成员
